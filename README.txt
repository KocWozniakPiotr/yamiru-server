# Yamiru Server. The mobile MMO game server with a custom profile authentication.

### current features:
- custom profile authentication through a PNG binary image backup
- creating account on the SQL-Server side
- reconnecting session 
- banning system 
- storing sessions 
- forwarding server notifications back to the client app

### work in progress:
- player database

### upcomming:
- terrain generation
- item database 
- monster database 
- day and night cycle 
- trading / broker system
